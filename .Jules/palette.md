## 2024-04-30 - [Reusable Pattern] Inline Confirmation for Destructive Actions
**Learning:** Destructive actions (delete, void, discard) should use inline confirmations instead of modals to keep the user in context and reduce friction while maintaining safety.
**Action:** Use the `InlineConfirmation` component for all 'Delete' or 'Discard' triggers in forms and detail views.
